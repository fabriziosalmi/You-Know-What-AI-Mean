# Contributing

This is a documentation repository. The substance lives in `README.md` and the linked artifacts in *What came next*. There is no source code to compile.

## Ways to contribute

- **Open an issue** if you find a factual error (a misattributed framework, a broken link, a misquoted citation), spot a typo, or want to suggest a missing perspective.
- **Open a PR** for direct text fixes (typos, broken Markdown, broken links).
- **Submit your own essay** if you want to write a section on a principle, trade-off, or case study the current draft handles weakly. Open an issue first to align on scope.

## What this guide is not for

- Promotion of unrelated tools or platforms.
- Long abstract takes without grounding in real artifacts or incidents.
- Re-introduction of the cut content (the UAEAI formula, sci-fi sections, 100-question scoring). The reasons they were cut are documented in commit history.

## Style

- Plain English, short sentences, concrete examples.
- One short quote per external source, in quotation marks.
- Cite frameworks (OECD, EU AI Act, NIST, UNESCO) by article or function name when possible.
- No emoji unless it carries meaning.
